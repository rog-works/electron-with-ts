Electron with Typescipt
===

# Requirements

* Node.js

# Dependencies

* Electron: 1.8.2
* TypeScript: 2.7.2
* Vue.js: 2.5.14
* Webpack: 4.0.1
* Font Awesome: 5.0.6
* Normalize CSS: 5.0.0

# Usage

## Install

```bash
$ npm install --no-optional
```

## Run

```bash
$ npm run bundle
$ npm run start
```
